**WebSite schema** — with name, url, and SearchAction (for sitelinks search box)

**Organization schema** — name, logo, url, sameAs (social media links)

**WebApplication schema** — like this:
```json
{
  "@context": "https://schema.org",
  "@type": "WebApplication",
  "name": "[MyAppName]",
  "url": "https://mywebsite.com",
  "applicationCategory": "SocialNetworkingApplication",
  "operatingSystem": "Web, Android, iOS",
  "offers": { "@type": "Offer", "price": "0", "priceCurrency": "USD" },
  "description": "Free chat and dating app to meet new people online."
}
```

**FAQPage schema** — create a FAQ page with 8–10 questions like "Is this chat app really free?", "How do I start chatting?", "Is it safe?" and add FAQPage JSON-LD so Google shows rich results

**BreadcrumbList schema** on inner pages
